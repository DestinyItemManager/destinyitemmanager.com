什麼是 Destiny Item Manager (DIM)?

您的物品隨時隨地。

忘了從保險庫中拿走你最喜歡的手炮？只需單擊並將其拖動到您的活動角色，同時飛入您的活動。如果您在塔中或在太空中閒逛，請直接裝備它!

創建完美的裝載。

使用 DIM 強大的加載優化器來選擇您想要的統計數據和模組，並讓 DIM 找到完美的項目集以助您一臂之力。當您創建指定它們的裝載時，DIM 甚至可以為您裝備您的模組!

快速找到您需要的東西。

DIM 中強大而快速的搜索查詢系統可以快速輕鬆地找到您要查找的任何內容。只需開始輸入您想要的內容，然後讓自動完成功能完成剩下的工作。您甚至可以創建複雜的搜索查詢來識別特定的護甲卷，並保存查詢以供重複使用! 查看我們的搜索視頻。

內置建議。

DIM 與社區專家合作，策劃了一份物品卷的願望清單，如果它們進入你的庫存，就會被豎起大拇指標記。單擊任何項目的名稱以查看其所有推薦卷的列表。您甚至可以使用您最喜歡的物品卷創建自己的願望清單，以便在 DIM 中使用!

高效發揮。

DIM 的進度頁面和賞金指南將幫助您跟踪下一步該去哪里以及需要攜帶哪些物品來完成最多的賞金任務。

隨處訪問 DIM。

DIM 是一種漸進式 Web 應用程序 (PWA)，可在任何具有受支持瀏覽器的設備上運行。在您的計算機、平板電腦或手機上使用 DIM！也可以從支持 PWA 的瀏覽器（如 Chrome、Edge 和 Safari）安裝 DIM，以獲得類似應用程序的體驗。
